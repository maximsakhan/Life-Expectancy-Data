Life Expectancy Prediction¶

The aim of this study is to investigate the determinants of life expectancy with the purpose of identifying the most significant factors that impact it. The selected factors will then be used as predictors in a predictive model. The analysis will involve a two-stage approach consisting of variable selection algorithms followed by regression algorithms, including LASSO and RIDGE.

To initiate the analysis, we will first import the dataset into R and proceed to examine its underlying structure.

For each predictor } X_j \text{ not in } S, \text{ fit the model } M_k \text{ with the predictors in } S \text{ plus } X_j \\ & \quad \text{and calculate a performance metric (e.g., AIC, BIC, or p-value).} \\ & \quad \text{b. Select the predictor } X_j \text{ that improves the performance metric the most and add it to } S. \\ 4. & \text{Continue the loop until a stopping criterion is met (e.g., no further improvement in the performance metric).} \\ 5. & \text{The final model contains the selected predictors: } M_{\text{final}}. \end{align*} $$

# Handle missing data
life_expectancy_clean <- na.omit(life_expectancy)

# Split the data into training and testing sets
set.seed(123)
trainIndex <- createDataPartition(life_expectancy_clean$Life.expectancy, 
                                   p = 0.8, list = FALSE)
trainData <- life_expectancy_clean[trainIndex, ]
testData <- life_expectancy_clean[-trainIndex, ]

# Define the predictor variables
predictors <- names(life_expectancy_clean)[4:19]

# Set the maximum number of predictors to consider
max_n <- length(predictors)

# Create an empty data frame to store the RMSE values
rmse_df <- data.frame(nvmax = 1:max_n, RMSE = NA)

predict.regsubsets = function(object, newdata, id, ...) {
    form = as.formula(object$call[[2]])
    mat = model.matrix(form, newdata)
    coefi = coef(object, id = id)
    mat[, names(coefi)] %*% coefi
}

# Run the stepwise forward algorithm for different values of nvmax
for (i in 1:max_n) {
    model <- regsubsets(Life.expectancy ~ ., data = trainData,
                      nvmax = i, method = "forward")
    pred <- predict(model, testData, i)
    rmse_df[i, "RMSE"] <- RMSE(pred, testData$Life.expectancy)
}
